# 📘 Student–Teacher Appointment Booking System

## 🔍 Overview

The **Student–Teacher Appointment Booking System** is a web-based application designed to simplify communication between students and teachers. It allows students to book appointments with teachers, while teachers can manage and respond to those requests efficiently.

The system provides separate dashboards for **students, teachers, and admins**, ensuring organized access and role-based functionality.

---

## Key Features

* **User Authentication**
  * Student and teacher registration & login
* **Appointment Booking**
  * Students can request appointments with teachers
* **Teacher Dashboard**
  * Teachers can view and manage appointment requests
* **Admin Dashboard**
  * Admin can monitor users and system activity
* **User Profiles**
  * Users can view and update their profile information
* **Responsive UI**
  * Clean and simple interface using HTML, CSS, and JavaScript

## Installation and Setup

This is a frontend-based project, so no complex installation is required.

**Requirements:**
A modern web browser (Chrome, Edge, Firefox, etc.)
Code editor (optional, e.g., VS Code)
Steps:
Download or extract the ZIP file.

**Open the project folder:**
student-teacher-app/
Ensure all files (HTML, CSS, JS) are in their correct directories.

---

## Steps to Run the Website:

* Open the project folder.
* Double-click on index.html
(or right-click → Open with browser)
* Navigate through the website:
* Register a new user
* Login using credentials
* Access dashboards based on role
